Random Word API

Current Languages:

EN - English

NL - Dutch

How to use:

/api/v1/resources/words?lang=EN&amount=5

There are 2 queries:

  • lang=EN - to specify the language.
  • amount=5 - the amount of random words. (max. 1000)

Example response:

{ "data": { "language": "EN", "amount": 1, "words": [ { "index": 1702, "word": "accommodate" } ] } }
